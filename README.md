# Personal Portfolio – React

A modern, responsive personal portfolio website built with React to showcase my projects, skills, and experience as a developer.

## 🚀 Overview
This project is my personal portfolio website. It acts as a single place to present:
- Who I am
- What I can build
- The technologies I work with
- How to contact me

The goal is not just design, but **clear communication of my skills through real projects**.

## 🧠 Tech Stack
- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite / Create React App *(whichever you used)*
- Git & GitHub



## ✨ Features
- Fully responsive layout
- Component-based architecture
- Project showcase section
- Skills and tech stack section
- Contact / social links
- Clean and minimal UI
